## ğŸ§© Problem Statement

Road accidents are a critical public safety issue causing thousands of deaths, injuries, and economic losses every year. However, accident data is often scattered across multiple fields and does not provide immediate clarity to policy makers.

This project solves that problem by building a **comprehensive, interactive dashboard** that answers:

- When do most accidents occur?
- Which road types and surfaces are most dangerous?
- What environmental conditions influence accident severity?
- How many casualties occur and at what severity levels?
- What patterns can help reduce road accidents?

The dashboard provides data-driven insights for efficient decision-making in traffic safety and enforcement planning.

- Backend sheet with pivots, helper tables, broken-down segments used by dashboard.

### **8. Dashboard**
- Final Professional Dashboard combining:
  - KPIs  
  - Monthly trends  
  - Donut charts  
  - Road type visual  
  - Road surface visual
